Flux Comics adalah website berbasis **Next.js** yang dibuat untuk menampilkan komik, manhwa, atau ilustrasi anime secara modern, cepat, dan responsif.

---

##  Tech Stack
- **Next.js (App Router)**
- **React**
- **Tailwind CSS**
- **TypeScript**
- **Next Font (Geist)**
- **Vercel Ready**

---

##  Features
-  Tampilan komik / manhwa modern
-  Fast rendering dengan Next.js
-  Responsive design (mobile & desktop)
-  Mudah dikembangkan
-  Siap untuk dark mode (opsional)
-  Support image optimization
